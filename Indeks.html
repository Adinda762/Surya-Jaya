    Tentang Kami

    Perusahaan kami adalah perusahan yang terletak di daerah Jakarta Selatan. Perusahaan ini berdiri sejak tahun 2007, pada awal mula sejak berdiri, perusahaan ini berbentuk Commanditaire (CV), namun sejak tahun 2020 telah berubah menjadi Perseroan Terbatas (PT).

    Bidang usaha Produk perusahaan kami adalah peralatan safety (safety equipment), tabung alat pemadam api (APAR) , fire hydrant system, fire alarm system, fire sprinkler system, komponen dan spare part serta asesoris fire hydrant, baju tahan api atau panas, selang, box APAR, fire, blanket.

    Cara Penggunaan Produk

    Tarik safety pin atau pin pengaman pada APAR, jangan sampai menekan tuas atas dan bawah secara bersamaan. Hal ini bisa membuat pin pengaman sulit dilepas.

    Pegang ujung selang APAR, kemudian arahkan dengan tepat ke sumber api. Memegang selang pada bagian tengah atau pengkalnya dapat membuat media pemadam api tidak terarah dengan tepat ke titik api.

    Tekan tuas secara penuh bertujuan agar bisa lebih cepat untuk mengeluarkan seluruh isi media di dalamnya, sehingga api bisa cepat padam.

    Gerakkan nozzle dari sisi kanan ke sisi kiri secara berulang. Langkah ini bertujuan agar penyebaran bahan pemadam api bisa merata dan kebakaran bisa dipadamkan dengan segera.
